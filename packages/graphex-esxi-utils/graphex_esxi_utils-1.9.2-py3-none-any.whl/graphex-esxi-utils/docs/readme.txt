This folder contains the documentation for this plugin. Edit the markdown files to make changes to the documentation.

After editing the markdown files, make sure to run (at least) 'make docs' to generate the updated html files. They are also generated on 'make build' and 'make all'.

The 'html' folder is populated with HTML files after one of: 'make docs', 'make build', or 'make all' is run. Do not edit the files in this folder, they are for viewing only.
