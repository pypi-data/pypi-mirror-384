# 测试文档标题

## 第一部分 - 基础概念
### 1.1 什么是XMind
XMind是一款思维导图软件，用于可视化思维过程。

### 1.2 主要功能
- 创建思维导图
- 项目管理
- 头脑风暴
- 会议记录

## 第二部分 - 高级功能
### 2.1 主题样式
支持多种主题样式和颜色配置。

### 2.2 导出选项
可以导出为多种格式：
- PDF文件
- 图片格式
- Word文档
- Excel表格

### 2.3 协作功能
- 团队共享
- 实时编辑
- 评论系统

## 第三部分 - 使用技巧
### 3.1 快捷键
常用快捷键列表：
- Ctrl + N: 新建
- Ctrl + S: 保存
- Ctrl + Z: 撤销

### 3.2 最佳实践
1. 保持层级清晰
2. 使用适当的颜色
3. 添加图标和标记
4. 定期保存工作
