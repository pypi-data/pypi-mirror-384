Extensión de l10n_es_facturae — pagos especiales
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Tipos especiales
~~~~~~~~~~~~~~~~

   - Cuando el modo de pago sea especial (13), se usará la cuenta
